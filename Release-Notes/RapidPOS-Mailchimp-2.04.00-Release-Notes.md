# Rapid POS Mailchimp Connector v2.04.00

**Release Date:** May 20th, 2026

## New Features

### Mailchimp Configuration Lookup Day
Added a lookup day setting in Mailchimp configuration for customer profile downloads.
Go to **Connectors > MailChimp > MailChimp Configuration**

## Enhancements

### Logging Improvements
Updated the Mailchimp Connector logging behavior to improve log retention and simplify troubleshooting.

- Enhances visibility into connector activity and processing
- Improves long-term log management and diagnostic capabilities

### Customer Profile Download Performance
Refactored the Mailchimp customer profile download process to improve performance and speed up data processing.

- Reduces processing time for customer profile synchronization
- Improves overall efficiency when downloading customer data
